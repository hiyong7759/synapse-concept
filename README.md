# Synapse — AI의 맥락 시냅스

**LLM의 "전체 히스토리 전부 주입" 구조를 뒤집어, 지금 이 질문에 공명하는 맥락만 선택적으로 연결하는 대화 레이어**

---

## 왜 만드는가

지금 AI 대화는 이렇게 작동합니다:

```
60턴 대화했으면 → 매번 60턴 전부를 인풋으로 넣는다
관련 있든 없든. 전부.
```

사람의 뇌는 이렇게 작동하지 않습니다. "인증 시스템 만들어줘"라고 하면 보안 경험은 활성화되고 오키나와 맛집 기억은 활성화되지 않습니다. 관련 없는 건 연결하지 않으므로 잡음이 없습니다.

현재 AI에는 이 **선택적 연결**이 없습니다. 컨텍스트 윈도우에 들어간 모든 정보를 동일한 무게로 참조합니다. 그래서:

- 앞 섹션은 잘 되는데 뒤 섹션은 개판이 됩니다
- 한 단어가 맥락 전체를 뒤집어버립니다
- 새 세션을 열면 패턴 자체가 달라집니다
- 잘 되던 시점으로 돌아갈 방법이 없습니다

**AI가 멍청해 보이는 순간들은 대부분 모델의 한계가 아니라 맥락의 문제입니다.**

## 무엇을 만드는가

Synapse는 뇌의 시냅스처럼 작동하는 AI 대화 레이어입니다:

- 모든 대화 턴을 항상 넣는 게 아니라, **지금 질문에 반응하는 맥락만 활성화**
- 자주 함께 효과적이었던 맥락 조합은 **연결 강화**
- 결과를 망쳤던 맥락은 **연결 약화**
- 쓸수록 연결이 좋아짐

### 핵심 구조: 듀얼 세션

```
세션 A (의도 정제)
  사용자와 가벼운 대화로 진짜 의도를 끌어냄
  → 구체화된 프롬프트 승인
        ↓
  컨텍스트 프리뷰
  벡터 검색으로 관련 맥락만 선별
  사용자가 검토·편집 가능 (턴 내부 부분 편집 포함)
        ↓
세션 B (클린 실행)
  세션 A의 지저분한 대화는 하나도 포함되지 않음
  정제된 프롬프트 + 검토된 맥락만으로 깨끗하게 실행
```
