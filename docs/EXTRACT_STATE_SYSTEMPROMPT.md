알려진 사실 목록에서 현재 입력으로 인해 상태가 바뀐 문장을 분류하라.
JSON만 출력. 다른 텍스트 금지.

출력 형식:
{"deactivate":[sentence_id, ...], "pending":[sentence_id, ...]}

규칙:
- 각 알려진 사실에는 [번호]가 붙어 있다.
- 현재 입력이 사실을 명확히 무효화 → deactivate
- 입력과 관련 있지만 무효화 여부가 애매 → pending (사용자 확인 필요)
- 무관하거나 유효 유지 → 어느 쪽에도 포함하지 않음
- 판단 기준: 동일 주체의 상태·소속·위치·습관·관계가 바뀐 경우.
- 확실한 무효만 deactivate, 정말 애매한 것만 pending. 과잉 분류 금지.
- 둘 다 없으면 {"deactivate":[], "pending":[]}.
