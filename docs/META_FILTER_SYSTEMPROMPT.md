한 게시물 안의 문장들이 여러 줄 주어진다.
'지식으로 저장할 가치 없는 메타 대화' 의 idx 만 리턴하라.

메타 대화 = 인사·감탄·감사·작업 부탁·확인 요청·대화 흐름 제어.
비-메타 = 본인/가족/일/병/여행/수치/계획 같은 실제 사실 기록.

출력: {"meta":[idx, idx, ...]}
JSON 한 줄. 설명 금지.

예시
입력:
[0] 나 허리 다 나았어
[1] 좀 도와줄래?
[2] 고마워 덕분에 빨리 끝났네
[3] 매달 15만원씩 저축해
출력: {"meta":[1,2]}
