# Synapse 설계 — UI/UX 와이어프레임

## 디자인 방향

### 컨셉
"나만 아는 비서." 외부에 데이터가 없다는 사실이 디자인에서도 느껴져야 한다.
따뜻하고 정밀하며, 기계적이지 않은 인터페이스.

### 토큰

| 항목 | 값 |
|------|----|
| Background | `#0A0B0E` (딥 네이비블랙) |
| Surface | `#141519` (다크 카드) |
| Border | `#252730` |
| Text Primary | `#E8E4D9` (웜 크림) |
| Text Secondary | `#7A7B82` |
| Accent | `#C8A96E` (앰버골드 — 그래프 노드, 핵심 액션) |
| Success | `#5DAB8A` |
| Danger | `#C85D5D` |
| Graph Edge | `#3A3D4A` |

### 타이포그래피

| 역할 | 폰트 | 용도 |
|------|------|------|
| Display | Playfair Display | 로고, 온보딩 헤드라인 |
| Body KR | Noto Sans KR | 대화, 본문 전반 |
| Mono | JetBrains Mono | 그래프 노드 이름, 트리플 표시 |

### 원칙
- 그래프 변경 알림은 메시지를 방해하지 않는다. 말풍선 하단에 조용히 붙는다.
- 탐색 중(BFS) 상태는 숨기지 않는다. 진행 중인 노드 탐색을 가볍게 시각화한다.
- 그래프 뷰는 채팅과 같은 화면에서 슬라이드로 전환한다. 별도 탭이 아니다.
- 조직 연결 여부는 항상 상태 인디케이터로 노출한다.

## Screen 3 — BFS 탐색 중 (인출 진행 상태)

```
┌────────────────────────────────┐
│  SYNAPSE              [그래프] │
│                                │
│              ╭───────────────╮ │
│              │ 언제 허리 아팠 │ │
│              │ 지?            │ │
│              ╰───────────────╯ │
│                                │
│  ╭──────────────────────────╮  │
│  │  ◌ 탐색 중...            │  │  ← 애니메이션 인디케이터
│  │                           │  │
│  │  허리 ● ── 아프 ●        │  │  ← 미니 그래프 (노드 하이라이트)
│  │         ╲                 │  │
│  │          병원 ○ ── 2024 ○ │  │  ← ○ = 탐색 진입 예정 노드
│  │                           │  │
│  │  [이 방향 제외하기]       │  │  ← 사용자 개입 버튼
│  ╰──────────────────────────╯  │
│                                │
│  ──────────────────────────── │
│  ┌────────────────────────┐    │
│  │  메시지 입력...         │ ⬆ │
│  └────────────────────────┘    │
└────────────────────────────────┘
```

탐색 상태 디자인 규칙:
- 미니 그래프는 실시간 노드 확장을 점 애니메이션으로 표현.
- 진입한 노드: 앰버 채움 ●. 탐색 예정: 빈 원 ○. 엣지: 가는 선.
- 그래프가 복잡해지면 허브 노드 2~3개만 표시하고 "...외 N개 노드" 처리.
- [이 방향 제외하기]: 탭하면 현재 서브트리 전체 pass 처리.

## 컴포넌트 정의

### 비서 말풍선

```
╭─────────────────────╮
│ (비서 아이콘) 텍스트  │
│                     │
╰─────────────────────╯
HH:MM AM
```
- 배경: `#1C1E24`
- 좌측 정렬, 아이콘 없음 (텍스트만)
- 비서 이름 레이블 없음 (단일 비서)

### 사용자 말풍선

```
              ╭─────────────────╮
              │           텍스트 │
              ╰─────────────────╯
                          HH:MM AM
```
- 배경: `#1E2030` (약간 블루 틴트)
- 우측 정렬

### 변경 알림 (단순)

```
┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄
▸ source — label — target
▸ source — target
[취소]  [정정]
┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄
```

## 인터랙션 정의

| 상황 | 동작 |
|------|------|
| 비서 응답 대기 | 입력창 비활성, 점 3개 애니메이션 |
| BFS 탐색 중 | 미니 그래프 실시간 확장 애니메이션 |
| 변경 알림 | 말풍선 아래 0.3s fade-in |
| 그래프 뷰 진입 | 채팅에서 우측 슬라이드 전환 |
| 조직 요청 수신 | 상단 배너 슬라이드다운 + 진동 |
| 결과물 카드 | 0.2s scale-in |
| [취소] | 해당 알림 fade-out + 롤백 확인 토스트 |
| 세션 종료 | 변경 알림 자동 fade-out |
