# Synapse Visualize — 지식 그래프 시각화

개인 지식 그래프를 인터랙티브 HTML로 생성하고 브라우저에서 연다.

## 실행 절차

### 1단계: 그래프 생성

```bash
python3 scripts/visualize.py
```

결과:
- `~/.synapse/graph.html` 생성
- 자동으로 기본 브라우저에서 열림

### 2단계: 결과 안내

```
그래프가 브라우저에서 열렸습니다.

기능:
- 노드 검색 (좌측 검색창)
- 도메인별 필터링 (그룹 토글)
- 노드 클릭 → 연결된 엣지 확인
- 다크/라이트 테마 전환
- 모바일 반응형 지원
```

### 브라우저에서 안 열릴 때

`--no-open` 옵션으로 파일만 생성:

```bash
python3 scripts/visualize.py --no-open
```

생성된 파일 경로를 사용자에게 안내한다:
```
그래프 파일: ~/.synapse/graph.html
브라우저에서 직접 열어주세요.
```

## 에러 처리

- DB가 없으면: 자동 초기화 후 빈 그래프 생성
- 노드가 0개이면: "저장된 노드가 없습니다. /save로 먼저 지식을 추가해주세요."
